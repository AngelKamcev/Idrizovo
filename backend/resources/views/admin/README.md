# Admin Panel - Идризово

Красиво дизајниран администрациски панел за управување со содржина на веб-страната на КПУ-КПД Идризово.

## Характеристики на Admin Panel

### 📊 Dashboard
- Преглед на вкупни активности, соопштенија, слики и производи
- Статус на системот (веб сервер, база на податоци, датотечен систем)
- Последни соопштенија
- Користена меморија и складишна просторот

### 🎯 Модули за Управување

#### 1. **Активности на Почетната Страна**
   - Управување со активностите што се прикажуваат во слајдерот
   - Додавање, уредување, бришење активности
   - Преглед на слајдер

#### 2. **Соопштенија**
   - Две картице: Нови соопштенија / Постари соопштенија
   - Управување со соопштенијата со статус (Објавено, Нацрт)
   - Датум на публикување
   - Брзо пребарување и филтрирање

#### 3. **Рачни Изработки**
   - Управување со производи (торби, резби, слики, керамика итн.)
   - Статус производи (На стану, Низ процес, Архивирано)
   - Филтрирање по категории
   - Grid приказ со хувер ефекти

#### 4. **Галерија**
   - Управување со сликите во галеријата
   - Качување нови слики
   - Приказ во grid формат
   - Хувер ефекти за брзи дејства (преглед, избришување)

#### 5. **За Нас**
   - Три картице: Основни информации / Одговорни лица / Миција и Визија
   - Уредување опис на установата
   - Управување со одговорни лица (додавање, уредување, бришење)
   - Основни информации (година, број местa, адреса)
   - Лого слика

#### 6. **Главни Активности**
   - Управување со сите активности во установата
   - Статус (Активни, Неактивни)
   - Датум на создавање
   - Брзо пребарување

## 🎨 Дизајн особености

- **Современ UI** - користи Tailwind CSS за леп и одговорен дизајн
- **Лесна навигација** - sidebar со јасна структура на модули
- **Интуитивни контроли** - јасни копчиња и форми
- **状атус индикатори** - визуелни показатели за статус со бои
- **Табели и картице** - различни прикази за различни типови содржина
- **Grid приказ** - за производи и слики
- **Хувер ефекти** - интерактивна повратна информација

## 📱 Адаптивност

Панелот е полностоумно адаптиван:
- Мобилни уреди (мобилни телефони)
- Таблети
- Лаптопи и десктопи

## 🔧 Технологии

- **Tailwind CSS** - за стилизирање
- **Font Awesome** - за иконите
- **Blade Template Engine** - за рендерирање
- **Laravel** - backend фремворк

## 📍 Приход до Admin Panel

Панелот е дизајниран само (UI дизајн - без функционалност). За приход:

```
/backend/resources/views/admin/dashboard.blade.php
/backend/resources/views/admin/activities.blade.php
/backend/resources/views/admin/soopstenija.blade.php
/backend/resources/views/admin/izrabotki.blade.php
/backend/resources/views/admin/gallery.blade.php
/backend/resources/views/admin/aboutus.blade.php
/backend/resources/views/admin/index_activities.blade.php
```

## 🎭 Цвета и стили

- **Примарна боја**: Сина (#0b5ecc)
- **Секундарна боја**: Сива (за неутрални елементи)
- **Статус бои**: 
  - Зелена - активно, успешно
  - Жолта - во процес
  - Црвена - несуществува, грешка
  - Синя - нацрт, информација

## 📝 Забелешки

Овај админ панел е **само дизајнер UI** и не содржи функционалност. За интеграција со база на податоци и функционалност, потребно е додатно развој на backend логика.
